# 📘 **Requirement Timer — 项目文档（v1.0 -> v2.0 升级版）**

## 📝 1. 项目简介

**Requirement Timer** 是一个用于游戏策划（或任何需求管理角色）使用的 **本地桌面应用**，核心用于：

- 管理需求进度
- 查看任务推进天数
- 显示剩余天数（含逾期标红）
- 敏捷看板与列表双视图切换 
- 游戏化工作体验（XP与等级系统）
- 提供干净、现代的 Apple 风格 UI 加上丰富的扩展统计组件。

整个项目使用 **Python + PySide6 + SQLite** 开发，可单机使用，也未来可封装为 exe。

## 🌟 2. v2.0 核心新增功能汇总

### 💡 **情感化与游戏化体验**
- **策划灵感语录 (Maxim Footer)**：底部状态栏随机展示《游戏设计艺术》等行业经典名言，点击可手动刷新，随时补充脑洞！
- **项目 XP 养成系统 (Gamification)**：将需求完成转化为经验值，需求上线（归档）即可获得 XP 并升级。XP 进度条带有平滑的增长动画。（高优 100XP，中优 50XP，低优 20XP）

### 📊 **生产力与数据统计**
- **数据统计仪表盘 (Dashboard)**：基于 `matplotlib` 绘制的可视化看板。
  - **燃尽图 (Burndown Chart)**：直观展示项目待完成任务的剩余消耗曲线。
  - **模块精力雷达图 (Radar Chart)**：快速透视系统、UI、数值等各个模块的任务分布情况。
- **敏捷看板视图 (Kanban Board)**：加入看板/列表模式一键切换功能，提供类似 Trello 的卡片式拖拽视觉展示（各列依据“策划中、开发中”等状态排列）。
- **富文本/Markdown 备注**：新建与编辑任务对话框引入 `编辑 / 预览` 双栏设计，支持 Markdown 渲染（表格、加粗、代码块等），更易于沉淀复杂的需求思路。
- **全局搜索与导出**：顶栏新增实时搜索框，并提供一键导出全体任务到 CSV 的功能。

### ⚙️ **底层架构优化**
- **数据库连接安全解耦**：引入 Context Manager（`with get_db_connection()`），根绝 SQLite 死锁等异常。
- **配置文件外置**：将状态流、优先级选项及 XP 经验值权重解耦至独立的 `config.json` 文件中，避免硬编码。

## 🗂️ 4. 状态逻辑说明

### **所有状态：**

```
策划中
交互案
美术
开发中
待验收
验收完
测试回归bug
已上线（归档）
```

### **属于“已完成”的状态：**

```
验收完
测试回归bug
已上线（归档）
```

### **显示逻辑：**

| 状态                | 剩余天数显示   | 进度条       |
| ------------------- | -------------- | ------------ |
| 非完成状态 & 未过期 | 正常显示       | 依推进计算   |
| 非完成状态 & 已过期 | 红色「已逾期」 | 保持自动计算 |
| 完成状态（任意）    | 绿色「已完成」 | 强制100%     |

## 🧪 6. 数据库结构（SQLite）

表名：`task`

字段：

| 字段       | 类型                | 说明                      |
| ---------- | ------------------- | ------------------------- |
| id         | INTEGER PRIMARY KEY | 自增 ID                   |
| title      | TEXT                | 标题                      |
| module     | TEXT                | 模块，例如 AI、系统、数据 |
| version    | TEXT                | 版本号                    |
| status     | TEXT                | 状态                      |
| priority   | TEXT                | 高/中/低                  |
| plan_start | TEXT                | 开始日期 YYYY-MM-DD       |
| plan_end   | TEXT                | 截止日期                  |
| notes      | TEXT                | 备注                      |

**新增表名：`gamification`** (用于项目XP成长控制)
| 字段       | 类型                | 说明                      |
| ---------- | ------------------- | ------------------------- |
| key        | TEXT PRIMARY KEY    | 键值对 Key (如 'xp')      |
| value      | INTEGER             | 相关经验值                |

## 📦 9. 如何打包为 EXE（可选）

使用 PyInstaller：

```
pyinstaller --noconfirm --windowed --clean ^
  --add-data "data;data" ^
  --icon=icon.ico ^
  main.py
```

生成的 exe 在 `dist/main/` 下。
